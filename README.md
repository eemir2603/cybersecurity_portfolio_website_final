# Cybersecurity Portfolio Website — 


## About

A dark, terminal-themed portfolio showcasing my journey into cybersecurity and penetration testing — built from scratch while studying English Language & Literature at Near East University. Includes an interactive command bar you can actually type into (try `help`).

## Tech stack

- HTML5 / CSS3 (no frameworks)
- Vanilla JavaScript
- Hosted on [Netlify](https://netlify.com)
- Custom domain via Namecheap DNS

## Features

- Boot-sequence intro animation
- Functional command-line navigation (`whoami`, `skills`, `projects`, `roadmap`, `contact`, `github`, `linkedin`, `sudo`, `clear`)
- Responsive layout, no external JS dependencies
- Respects `prefers-reduced-motion`

## Sections

- **whoami** — who I am, what I'm doing
- **skills** — languages, Linux/terminal skills, cryptography basics
- **projects** — Caesar Cipher CLI, Bandit write-ups, and what's next
- **roadmap** — my certification path: BTK Akademi → Google Cybersecurity → eJPT → Security+ → OSCP
- **contact** — GitHub, LinkedIn

## Running locally

No build step needed — it's a single static file.

```bash
git clone https://github.com/eemir2603/cybersecurity_portfolio_website_final.git
cd cybersecurity_portfolio_website_final
open index.html
```

## Deployment

Auto-deploys to Netlify on every push to `main`.
